# Купе

Короткий детективный рассказ и материалы к нему.

Летняя ночь после бури. Медленный поезд Москва — Санкт-Петербург, купе на четыре полки, четверо пассажиров, севших на разных станциях. В середине пути буря валит дерево на рельсы, поезд встаёт в глуши без связи — и до рассвета один из попутчиков не доживает. Повесть ведётся от лица отставного офицера, в манере приключенческой прозы Стивенсона.
