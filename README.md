# Купе

Короткий детективный рассказ и материалы к нему.

Летняя ночь после бури. Медленный поезд Москва — Санкт-Петербург, купе на четыре полки, четверо пассажиров, севших на разных станциях. В середине пути буря валит дерево на рельсы, поезд встаёт в глуши без связи — и до рассвета один из попутчиков не доживает. Повесть ведётся от лица отставного офицера, в манере приключенческой прозы Стивенсона.

## Варианты повести
Один сюжет, рассказанный тремя разными голосами:

1. **[В манере Стивенсона](plans/draft/kupe.md)** — ретроспективная исповедь отставного офицера, приключенческая проза с предвестиями и образностью «Острова сокровищ».
2. **[В манере военных мемуаров](plans/draft/kupe-pokryshkin.md)** — трезвый разбор «боя» по этапам в духе мемуаров лётчика-аса («Познать себя в бою»): короткие фразы, самокритика, выведенные правила.
3. **[В манере криминального чтива](plans/draft/kupe-pulp.md)** — жёсткое нелинейное повествование с переставленным порядком глав: открывается на трупе, кольцом возвращается к развязке.
