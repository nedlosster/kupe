# «Купе» — рабочая папка рассказа

Детективный короткий рассказ. **Летняя ночь после сильной бури.** Ночной фирменный поезд Москва — Санкт-Петербург, который тащит неисправный тепловоз. Замкнутое купе, четверо пассажиров входят на разных станциях. В середине пути буря валит дерево на рельсы — поезд встаёт в глуши без связи.
